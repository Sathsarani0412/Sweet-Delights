# Sweet Delights Cake Shop Website

## Overview

Sweet Delights Cake Shop Website is a responsive web application designed to help customers browse cakes, customize cake orders, and place orders online. The system provides an easy-to-use interface for customers while storing order and inquiry information using an SQLite database.

## Features

### Home Page

* Attractive landing page showcasing featured cakes and promotions.
* Easy navigation to all website sections.

### About Page

* Information about the cake shop, services, and mission.

### Cakes Page

* Browse available cake designs and cake categories.
* View cake images and details.

### Custom Cake Page

Customers can create personalized cakes by selecting:

* Cake Theme
* Cake Size
* Cake Design
* Additional Decorations
* Special Extras and Customizations

### Order Page

* Place cake orders online.
* Submit customer details and order requirements.
* Store order information in the SQLite database.

### Contact Page

* Send inquiries and feedback to the cake shop.
* Customer messages are stored in the database.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* SQLite

## Database Functionality

The system uses SQLite to manage:

* Customer Orders
* Custom Cake Requests
* Contact Form Submissions

## Key Functionalities

### Custom Cake Builder

Allows customers to personalize cakes according to their preferences and requirements.

### Order Management

Stores and manages customer orders efficiently through database integration.

### Contact Management

Records customer inquiries and feedback for future communication.

## Project Purpose

This project was developed to demonstrate front-end web development, database integration, form handling, and customer order management functionalities within a real-world e-commerce scenario.
